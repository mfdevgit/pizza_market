### / about

Фронтенд-проект пиццерии. Сделан на **reactjs**, подгружает данные через бэкенд из **mongodb**, хранит и использует данные с помощью **redux**. Подробное описание будет позднее.

Промокоды:

-   **WELCOME**
-   **GOODWORK**
-   **PIZZALOVE**
-   **WATERPLEASE**
